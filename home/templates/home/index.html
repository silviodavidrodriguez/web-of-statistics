{% translate "Free Online Statistical and Machine Learning Tools" %}

{% blocktranslate trimmed %} Free online statistical analysis and machine learning tools: ANOVA, MANOVA, regression, classification, PCA, clustering, tree models, probability distributions, quality control charts and consumer sensory analysis. {% endblocktranslate %}

{% blocktranslate trimmed %} Web of Statistics is a free online platform for statistical analysis and machine learning. It provides calculators, visualization tools and predictive models for students, teachers, researchers and professionals who need quick, accessible and reliable results without installing additional software. {% endblocktranslate %}

{% blocktranslate trimmed %} The platform includes tools for hypothesis testing, data visualization, probability distributions, regression models, classification methods, ANOVA, quality control and multivariate analysis, all accessible directly from a web browser. {% endblocktranslate %}

{% translate "About the project" %}

{% blocktranslate trimmed %} Web of Statistics was created by Dr. Silvio D. Rodríguez, Professor of Applied Statistics at the Pontificia Universidad Católica Argentina (UCA), Buenos Aires. The goal of the project is to make statistical tools more accessible for education, research and professional data analysis. {% endblocktranslate %}

{% blocktranslate trimmed %} This is an academic and educational initiative. The tools are provided freely, and users are responsible for checking results and applying statistical methods appropriately. {% endblocktranslate %}

{% blocktranslate trimmed %} Web of Statistics is an open-source project. The complete source code is publicly available on GitHub, allowing users, educators and developers to inspect, contribute to and improve the platform. {% endblocktranslate %}
